# morning-news-digest

毎朝 **08:00 (JST)** に、世界のニュースを厳選し、自分用にやさしく変換して **Claudeのチャット**へ届ける仕組みの設定リポジトリ。

## 仕組み

1. 毎朝、Claudeの定期実行エージェントがこのリポジトリの [`news-rules.md`](./news-rules.md) を読む
2. ルールに従って世界のニュースをWeb検索で収集・厳選（5〜7本）
3. 日本語に変換・整形して、チャットに配信

## ルールの編集

配信内容を変えたいときは [`news-rules.md`](./news-rules.md) を編集して push するだけ。翌朝の配信から反映されます。

- 関心テーマを足す／減らす
- 本数や時刻を変える
- 文体・長さ・出力フォーマットを変える
- 避けたい話題・媒体を「除外・NG」に追記する

口語で書いても箇条書きでもOK（エージェントが解釈します）。
