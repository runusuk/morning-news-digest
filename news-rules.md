# 毎朝ニュースダイジェスト — ルール

> **このファイルが唯一の設定です。** ここを編集すると、翌朝以降の配信に自動で反映されます。
> 毎朝 **08:00 (JST)** に自動実行され、このルールに従って世界のニュースを厳選・変換し、**Claudeのチャット**に届けます。
> ルールは思いついた順に自由に追記・編集してOK（箇条書きでも口語でも、エージェントが解釈します）。

## 2. 関心テーマ（何を拾うか）

幅広く世界を見たい。以下を主軸に、偏らないようバランスを取る。

- **国際政治・地政学** … 各国政治、外交、紛争、安全保障、選挙
- **経済・ビジネス** … マーケット、企業、金融、産業・通商の動向
- **テクノロジー・AI・科学** … IT、AI、研究、宇宙、医療・技術
- **社会・文化・環境** … 社会問題、文化、気候・環境、人権

> 重み付けの目安：4分野からなるべく満遍なく。ただし「その日に世界的に本当に重要な出来事」があれば分野バランスより重要度を優先する。

### 定点ウォッチ（毎回必ず掘る）

その日の大ニュースの有無に関わらず、下記の領域は**毎回必ず能動的に検索し、最低1本は取り上げる**。

- **防衛テック / 国家安全保障テック**：Anduril、Palantir、Shield AI など「軍事・防衛 × シリコンバレー」の動き（契約・受注、製品、自律兵器、調達、規制、資金調達）
- **Peter Thiel と周辺人脈**：Peter Thiel本人、Founders Fund、Palmer Luckey（Anduril創業者）、いわゆる「ペイパル・マフィア」、Thiel系の投資・政治的動き
- **関連テーマ**：dual-use（軍民両用）技術、監視・インテリジェンス、テック業界と政府・軍の結節点、テクノ・リバタリアン的潮流

- この枠は5〜7本のうち1本として数えてよい（重要ニュースが多い日は最大8本まで可）。
- 24時間以内に目立った動きが無ければ、直近の重要な進展を1本拾う。無理に誇張せず、捏造はしない。出典は信頼できる媒体で。

## 3. 厳選ルール（どう選ぶか）

- **重要度・影響範囲**で選ぶ。一過性の話題より、世界や日本に波及する出来事を優先。
- **信頼できる媒体だけを使う**（下記「信頼できるニュースソース」を優先）。**米国・欧州・中東**を主軸にカバーする。
- 1社のスクープを鵜呑みにせず、**複数ソースで裏が取れている**ものを優先。
- 速報の断片より、**背景まで分かる**ものを選ぶ。
- 重複・続報は1本にまとめる。
- 日本固有の国内ニュースは、世界的な文脈があるものを中心に（純ローカルは原則外す）。
- ゴシップ・芸能・センセーショナルなだけの話題は拾わない。

### 信頼できるニュースソース（米・欧・中東を中心に）

信頼性の高い媒体を優先的に使い、地域・立場が偏らないよう複数を組み合わせる。

- **通信社（最優先・裏取りの軸）**：Reuters / AP通信 / AFP / Bloomberg
- **米国**：The New York Times / The Washington Post / The Wall Street Journal / NPR / Politico / Axios / The Atlantic
- **欧州**：BBC / The Guardian / Financial Times / The Economist / Deutsche Welle (DW) / Le Monde / El País / Politico Europe
- **中東**：Al Jazeera / The Times of Israel / Haaretz / Middle East Eye / Al-Monitor / The National (UAE) / Arab News

※同じ出来事は、立場の異なる複数ソース（例：中東なら Al Jazeera と The Times of Israel）で照合し、中立に要約する。
※ここに載っていない媒体でも、定評ある報道機関なら可。タブロイド・まとめサイト・出所不明のSNS情報は使わない。

## 4. 変換ルール（どう書くか）← ここが「ぼく用」の核

- **知識のある読者（知識人）向け**でよい。無理にやさしくせず、専門用語や硬めの表現もそのまま使ってよい。用語の逐一の補足は不要（理解に必要な背景だけ簡潔に添える）。
- 各ニュースは「**何が起きたか → なぜ重要か / 背景 → だから何**」の流れで。
- 中立・事実ベース。煽らない。断定しすぎない（不確実なことは「〜とされる」等）。
- **構造的・歴史的な背景**を一言添えられるなら添える（点ではなく流れで理解できるように）。
- 1本あたり要約2〜3文 ＋ 背景1文。長くしない。
- 必ず**出典リンク**を付ける。

## 6. 除外・NG

- **芸能・エンタメは全面的に除外**（ゴシップ、有名人の私生活、映画・音楽・テレビの新作/興行など）
- **スポーツは全面的に除外**（試合結果・移籍・大会など）。ただし出来事の本質が国際政治・経済・社会である場合（例：大型大会を巡る国家間の対立）は、その文脈のニュースとして扱ってよい。
- タブロイド・まとめサイト・出所不明のSNS情報、広告的・PR記事
- 真偽不明・単一ソースのみの煽り記事
- 過度にグロテスク／不安を煽るだけの内容
- （ここに避けたい話題・媒体があれば追記）

## 7. 毎朝の手順（エージェント向け・通常は編集不要）

1. このファイルの最新版を読み、ルールを把握する。
2. 過去24時間の世界のニュースを、上の4テーマ横断で **Web検索で複数収集**する（信頼できる国際メディア中心）。**加えて「定点ウォッチ」の領域（Anduril / Palantir / Peter Thiel / Founders Fund 等）も毎回必ず個別に検索する。**
3. 「3. 厳選ルール」で **5〜7本に絞る**（うち「定点ウォッチ」から最低1本を必ず含める）。
4. 「4. 変換ルール」で日本語に変換し、「5. フォーマット」に整える。
5. 各本に**出典リンク**を必ず付ける。
6. 完成したダイジェストを、その実行の**最終メッセージとしてチャットに出力**する（これが配信物）。
